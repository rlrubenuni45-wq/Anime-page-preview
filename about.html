About AnimeWorld

AnimeWorld is a simple anime-theme website created for anime fans who want to explore popular anime series, characters, and genres in one place.

This website is designed for beginners and focuses on clean design, easy navigation, and a fun anime atmosphere. Whether you enjoy action, romance, fantasy, or comedy anime, AnimeWorld has something for you.

AnimeWorld was created as a learning project to practice web design using HTML and CSS. More features and pages will be added in the furture.

Why AnimeWorld?

  • Simple and beginner-friendly
  • Information about popular anime
  • clean and easy navigation
  • built with HTML & CSS
